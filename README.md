# MaidMic 🎤✨

**虚拟麦克风 · 变声引擎 · 实时 DSP · 开源**

MaidMic 是一个开源的 Android 虚拟麦克风应用，搭载自研 **Echio 变声引擎**（C/C++），
支持实时音频 DSP 处理：增益、均衡器、混响、变调、共振峰偏移、失真、回声。

## 特性

### 🎛 三引擎切换
| 引擎 | 说明 |
|------|------|
| **直通模式** | 音频不经处理直接透传，零延迟 |
| **Echio 均衡** | 增益 + 低音/高音 shelving + 变调 + 混响 |
| **频响曲线** | HxCore 风格 10 段 Biquad Peaking EQ + 前级预渲染 |

### 🎵 实时 DSP 效果
- **均衡器** — 低音/高音 shelving 或 10 段图形均衡
- **变调** — ±12 半音，线性插值重采样
- **混响** — 简单延迟线，干湿混合比可调
- **共振峰偏移** — Shelving filter 补偿，变调更自然
- **失真** — 软削波 waveshaping，机器人/恶魔声
- **回声** — 反馈延迟线，最长 500ms，衰减可调

### 🎯 10 个变声预设
原声 · 增强 · 低沉 · 萝莉 · 大叔 · 混响 · **机器人** · **恶魔** · **花栗鼠** · **幽灵**

### 📊 8 个频响曲线预设
JBL Flip · PartyBox · 环绕声（L/R） · 人声增强 · 耳机 Hi-Fi · 低音增强+

### 📱 HXAudio 风格 UI
- **Canvas 频响曲线图** — 动态度量网格 + 填充曲线
- **设备自动检测** — 有线耳机/蓝牙/扬声器状态栏
- **旁路开关** — 一键直通，不丢失设置
- **折叠效果器面板** — 核心控件始终可见，高级效果可折叠
- **测试变声** — 录音 3 秒 → 引擎处理 → 回放验证

### 🔌 三路虚拟麦克风
| 方案 | 权限 | 说明 |
|------|------|------|
| Root AudioFlinger | Magisk root | AudioRecord + AudioTrack 环回，SCHED_FIFO 优先级 |
| Shizuku AAudio | Shizuku | 非 root，推荐方案 |
| 无障碍服务 | 无障碍权限 | 最兼容，无需 root |

### 其他
- Material 3 暗色主题（Jetpack Compose）
- 前台保活服务
- 状态持久化（EQ 参数、引擎选择跨重启保留）
- 开源鸣谢页面

## 构建

CI 自动构建 APK，推送到 `main` 分支即可触发。
可在 [Actions](https://github.com/suer781/MaidMic/actions) 页面下载 `.apk` 文件。

### 本地构建

```bash
cd maidmic-app
./gradlew assembleDebug
```

### 依赖

- Android SDK 34
- NDK (CMake 3.22+)
- JDK 17
