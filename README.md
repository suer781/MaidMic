# MaidMic 🎤✨

**虚拟麦克风 · 变声 · Echio 引擎 · 开源**

> "数字时代声卡，女仆般温柔驾驭你的声音"

MaidMic 是一个开源的 Android 虚拟麦克风应用，搭载自研 **Echio 变声引擎**（C/C++），
支持实时音频 DSP 处理：增益、低音、高音、混响、变调。

---

## 特性

- **Echio 变声引擎** — C/C++ 原生实时音频 DSP 处理（NEON 可优化）
- **EQ 调节面板** — 6 种预设一键切换（原声/增强/低沉/萝莉/大叔/混响），滑块实时调参
- **三路虚拟麦克风方案** — Root AudioFlinger / Shizuku AAudio / 无障碍服务
- **Lua 插件系统** — UGC 插件运行时（沙箱隔离、权限分级）
- **模块链编辑器** — 线性或 DAG 模式编排 DSP 处理顺序
- **状态持久化** — EQ 设置退出重进不丢失
- **前台保活** — 通知栏常驻服务，防止进程被回收
- **Material 3 暗色主题** — Jetpack Compose 构建

## 构建

CI 自动构建 APK，推送到 `main` 分支即可触发。
可在 [Actions](https://github.com/suer781/MaidMic/actions) 页面下载 `.apk` 文件。

### 本地构建

```bash
cd maidmic-app
./gradlew assembleDebug
```

### 依赖

- Android SDK 34
- NDK (CMake 3.22+)
- JDK 17

## 子项目

| 目录 | 语言 | 说明 |
|------|------|------|
| `maidmic-app` | Kotlin | Android App (Jetpack Compose UI) |
| `maidmic-engine` | C/C++ | Echio 变声引擎（DSP 处理核心） |
| `maidmic-lua` | Lua | 插件系统运行时 |
| `maidmic-p2p` | Go | 插件市场 GitHub 索引同步 |
